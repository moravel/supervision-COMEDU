
Installation rapide sur Docker
### Comment déployer le serveur sur une nouvelle machine avec cette archive :
1. Extraire l'archive `deploy-server.zip` sur la machine cible.
L'archive contient l'essentiel :
- **`docker-compose.yml`** : Fichier chef d'orchestre pour démarrer le serveur.
- **`server-node/`** : Le code source de l'application, les assets (`public/`), et surtout le `Dockerfile` pour construire l'image.
- **`README.md`**, **`DOCUMENTATION.md`** et **`DEPLOIEMENT_SERVEUR.md`** : Pour que la personne en charge du déploiement ait toutes les informations. 
2. Ouvrir un terminal dans le dossier extrait.
3. Lancer la commande suivante (elle va télécharger Node.js, installer les dépendances et allumer le serveur) :
   docker compose up -d --build
   ```
4. Le serveur sera disponible immédiatement sur le port **3001** (comme configuré dans le `docker-compose.yml`)

Erreurs Docker :
# Arrêter tous les conteneurs en cours d'exécution
sudo docker stop $(docker ps -q)

# Supprimer tous les conteneurs (forcé avec -f)
sudo docker rm -f $(docker ps -aq)

Sinon

# Supprimer tous les conteneurs

sudo docker compose down`.
